------------------------------------
Replace <your-username> with your GitHub username:

Visit your repository URL. You should see:

================================================================================
  HOW TO RUN THE APPLICATION
================================================================================

OPTION A: RUN LOCALLY
---------------------
1. Install Python 3.10+ if not already installed

2. (Optional) Create a virtual environment:
       python -m venv venv
       source venv/bin/activate        # macOS/Linux
       venv\Scripts\activate           # Windows

3. Install dependencies:
       pip install -r requirements.txt

4. Run the application:
       streamlit run app.py

5. Open the URL shown in terminal (usually http://localhost:8501)


OPTION B: RUN ON GOOGLE COLAB (RECOMMENDED FOR DEMO)
------------------------------------------------------
1. Go to https://colab.research.google.com
2. Upload the file: EIA_Research_Assistant_Colab.ipynb
3. Upload the zip file: eia-streamlit.zip (to the Colab file browser)
4. Run all cells in order
5. A public URL will appear — click it to open the app
6. If localtunnel asks for a password, enter the IP shown in the output

Note: The Colab notebook includes all steps: install, unzip, test, launch.


OPTION C: DEPLOY ON STREAMLIT CLOUD (PERMANENT URL)
----------------------------------------------------
1. Push your code to GitHub (Steps 1-4 above)
2. Go to https://share.streamlit.io
3. Sign in with your GitHub account
4. Click "New app"
5. Select your repository: eia-energy-research-assistant
6. Branch: main
7. Main file path: app.py
8. Click "Deploy"
9. Your app will be live at: https://your-app-name.streamlit.app
